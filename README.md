# Experimental agent skills and agents

This is a work in progress and a build-in-public snapshot. It is for our own use first; you are welcome to look around. Nothing here is a product, a service, or a guarantee of safety or fitness for your use case.

**This repository contains both skills and agents.** Skills are reusable “how-to” instructions for an assistant (procedures, patterns, and guidance). Agents are role-style markdown definitions (who is responsible for what, and how work hands off). Together they are meant to be copied, adapted, or read for ideas; they are not a support contract.

**Two especially useful skills in this repo** if you want to build similar material yourself:

1. **[Agent creator](skills/agent-creator/SKILL.md)** — A blueprint for writing lean `*-agent.md` files: responsibilities, how the role decides, outputs, and completion and handoff.

2. **[Agent skill creator](skills/agent-skill-creator/SKILL.md)** — A blueprint for authoring agent skills: a tight `SKILL.md`, references for depth, and consistent structure.
