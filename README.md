# Experimental agent skills and agents

This is a work in progress and a build in public snapshot. We use it ourselves first; you are welcome to look around. Nothing here is a product, a service, or a guarantee that it is safe or fit for your use case.

**This repository contains both skills and agents.** Skills are reusable instructions for how an assistant should work (procedures, patterns, and guidance). Agents are role style markdown definitions: who is responsible for what, and how work hands off. You can copy, adapt, or read them for ideas.

**The flagship skill in this repo** is **[Agent skill creator](skills/agent-skill-creator/SKILL.md)**. It is the blueprint we use to create the rest: a tight `SKILL.md`, references for depth, and consistent structure. Start there if you want to build similar material yourself.
