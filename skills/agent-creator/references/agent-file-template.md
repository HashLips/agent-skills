# Agent File Template

## Summary

- Use this as the canonical layout for `*-agent.md` files.
- Keep files concise, list-first, and operational.

## Size and readability defaults

- **Target size** — 250-500 words.
- **Bullet density** — Prefer 4-7 bullets per major section.
- **Line density** — Prefer one-line bullets unless precision needs two.
- **Style** — Minimal emphasis; headings and lists carry structure.
