# Editorial Principles

## Preserve

- Meaning, facts, numbers, names, domain terms
- Author register, humor, formality
- Intentional repetition for effect
- Regions in [preserve-regions.md](preserve-regions.md)
- Fixed literals (official hyphenated names, URLs, user-marked blocks)

## Change (when flagged)

Smallest edit that breaks the **pattern**.

| Pattern | Fix |
| --- | --- |
| Hyphen join in copy | Separate words or rephrase |
| Em dash chain | Sentences, comma, or period |
| Semicolon stack | Separate sentences |
| Connector cluster | Drop or merge; one pivot |
| Vague phrasing cluster | Plain words already in the piece |
| Uniform rhythm | Vary one sentence length or opening |
| Summary-only ending | End on a concrete detail |
| Triple explain | Cut the restatement |

## Word choice

- Reuse words from the same piece.
- Prefer plain verbs and nouns when fixing a cluster.
- Do not add slang or hype the author did not use.

## Forbidden

- New hyphen joins in output
- Fake typos or forced casualness
- Thesaurus parade
- Rewriting quotes, stats, or marked preserve regions
- Changing the author's core argument or facts

## Voice check

Read before and after samples. Same person? Revert if flatter or more corporate.

## Change volume

Light ≤ 10% of sentences; standard ≤ 20%; heavy only if user asks.
